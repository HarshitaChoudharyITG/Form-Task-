A class is like a template or blueprint that tells JavaScript how to make an object. The object will have some properties (things it can have) and methods (things it can do).

For example :- let's imagine we're making cars. We want to make a bunch of cars, and each car has a color and a model, and each car can drive and stop.

Creating a Car Class

EXAMPLE :- // This is the Car blueprint (class) class Car { // The constructor is like a "setup" function that creates a new car constructor(color, model) { this.color = color; // This car has a color this.model = model; // This car has a model } // This is a function (method) that tells the car to drive drive() { console.log(`${this.model} is driving!`); } // This is another function (method) that tells the car to stop stop() { console.log(`${this.model} has stopped.`); } } // Now we use the class to create (make) a new car let myCar = new Car("red", "Toyota"); // This is a red Toyota // We can use the methods of this car myCar.drive(); // Output: Toyota is driving! myCar.stop(); // Output: Toyota has stopped.

Explaination
  • class Car { ... } is the blueprint. It says what a car can do (like drive and stop) and what a car has (like color and model).
  • constructor(color, model) is where we set up the car when we make it. We give it a color and model when we create a car.
  • Methods like drive() and stop() are actions that the car can do.

Summary of Basic JavaScript Class Concepts:

Class: A template (blueprint) for creating objects. Constructor: The special function inside the class that sets up the object. Properties: Things an object has, like color or model. Methods: Things an object can do, like drive or stop.

CLASS INHERITANCE & GETTER-SETTER

In JavaScript, classes and inheritance allow you to create objects and reuse code through a hierarchical relationship. Additionally, getters and setters are special methods that allow you to define how to access and update the properties of an object. 1. Class Inheritance: Inheritance allows one class to inherit properties and methods from another class. This is useful when you want to create new classes that share common functionality. Example: Class Inheritance // Parent class (Base class) class Animal { constructor(name, type) { this.name = name; this.type = type; } // Method in the base class speak() { console.log(`${this.name} makes a sound.`); } } // Child class (Derived class) inheriting from Animal class Dog extends Animal { constructor(name, type, breed) { // Call parent class constructor super(name, type); this.breed = breed; } // Overriding the speak method speak() { console.log(`${this.name} barks.`); } // New method specific to Dog fetch() { console.log(`${this.name} is fetching the ball.`); } } // Create an instance of Dog let myDog = new Dog("Buddy", "Mammal", "Golden Retriever"); myDog.speak(); // "Buddy barks." myDog.fetch(); // "Buddy is fetching the ball."
Explanation: Animal is the parent class with a constructor that initializes name and type properties, along with a speak method. Dog extends Animal, inheriting its properties and methods, and also has its own method fetch. super() is used to call the constructor of the parent class Animal when creating an instance of Dog.
2. Getters and Setters: Getters and setters are special methods that allow you to define how to get and set the value of an object's property. Getter: Allows you to access the value of an object property. Setter: Allows you to modify the value of an object property. Example: Getters and Setters class Person { constructor(firstName, lastName) { this.firstName = firstName; this.lastName = lastName; } // Getter get fullName() { return `${this.firstName} ${this.lastName}`; } // Setter set fullName(name) { let parts = name.split(" "); this.firstName = parts[0]; this.lastName = parts[1]; } } let person = new Person("John", "Doe"); console.log(person.fullName); // "John Doe" (using the getter) // Using the setter to update the full name person.fullName = "Jane Smith"; console.log(person.firstName); // "Jane" console.log(person.lastName); // "Smith" Explanation: Getter (fullName): This method allows you to retrieve the full name of the person by combining firstName and lastName. Setter (fullName): This method allows you to set a new full name, and it splits the name into firstName and lastName based on the space.
Summary: Inheritance allows you to reuse code from a base class in a derived class. Getters and Setters allow you to control how properties are accessed and modified.
