JavaScript Object Properties

1. KEY Points

Properties are the most important part of JavaScript objects.

Properties can be changed, added, deleted, and some are read only.

2.Accessing JavaScript Properties

The syntax for accessing the property of an object is:

1. // objectName.property let variable_name = object.key; Example :- const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" }; console.log(person.age); 2. //objectName["property"] let variable_name = object["key"]; Example :- const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" }; console.log(person["age"]); 3. //objectName[expression] let variable_name = object[key]; Example :- const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" }; let x = "firstname" console.log(person[x]);

3. Adding New Properties

You can add new properties to an existing object by simply giving it a value:

SYNTAX :- object.key = value; const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" }; person.nationality = "india"; console.log(perosn.fistname + "from" + person.nationality);

4. Deleting Properties

The delete keyword deletes a property from an object:

SYNTAX :- delete object.key const person = { firstName: "John", lastName: "Doe", age: 50, eyeColor: "blue" }; delete person.age; ANOTHER WAY To DELETE :- delete object["key"] const person = { firstName: "John", lastName: "Doe", age: 50, eyeColor: "blue" }; delete person["age"]; NOTE : The delete keyword deletes both the value of the property and the property itself and result in BOOLEAN Form. After deletion, the property cannot be used before it is added back again.

5. Nested Objects

Property values in an object can be other objects:

SYNTAX :- object = { object1 = { } } EXAMPLE :- myObj = { name:"John", age:30, myCars: { car1:"Ford", car2:"BMW", car3:"Fiat" } }

You can access nested objects using the dot notation or the bracket notation:

ACCESS SYNTAX :- a) Object.object1.key b) Object.object1["key"] c) Object["object1"]["key"] d) let p1 = "object1"; let p2 = "key"; object[p1][p2];
// Create nested Objects const myObj = { name: "John", age: 30, myCars: { car1: "Ford", car2: "BMW", car3: "Fiat" } } //Display Object console.log(myObj.myCars.car1); console.log(myObj.myCars["car2"]); console.log(myObj["myCars"]["car2"]); console.log(myObj[p1][p2]);
