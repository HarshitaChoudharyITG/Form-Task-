JavaScript Functions Notes

1. Function Declaration

Yeh sabse basic tareeka hai function define karne ka.

function keyword se shuru hota hai aur iska naam zaruri hota hai.

Isse pehle call karne pe bhi kaam karta hai kyunki yeh hoist hota hai..

Syntax: function functionName(parameters) { // Code to be executed } 

            function greet(name) {
                console.log("Hello, " + name + "!");
            }
            greet("Jack");

2. Function Expression

Yeh ek anonymous function ko ek variable mein store karta hai.

Isse pehle call nahi kar sakte kyunki yeh hoist nahi hota.

Syntax: const variableName = function(parameters) { // Code to be executed }; 

            const sum = function(a, b) {
                return a + b;
            };
            console.log(sum(5, 3));

3. Function Constructor

new Function() syntax se naya function banaya ja sakta hai.

Yeh runtime pe function create karta hai jo thoda slow ho sakta hai.

Syntax: const func = new Function('param1', 'param2', 'return param1 + param2;'); 

            const add = new Function('a', 'b', 'return a + b');
            console.log(add(2, 3));

4. Function Hoisting

Function declarations hoist hote hain, matlab unhe pehle call kar sakte hain.

Lekin Function Expressions hoist nahi hote.

sayHello(); // Works because of hoisting function sayHello() { console.log("Hello!"); } // greet(); // Error: Cannot access 'greet' before initialization const greet = function() { console.log("Hi!"); };

5. Self-Invoking Functions (IIFE)

Yeh functions turant execute ho jaate hain jab define hote hain.

Mostly anonymous hote hain.

Local scope maintain karte hain.

Syntax: (function() { // Code to be executed })(); Example: (function(name) { console.log("Welcome, " + name + "!"); })("Jack");

6. Functions Can Be Used as Values

Functions ko variables mein store kar sakte hain, dusre functions ko pass kar sakte hain, ya return kar sakte hain.

function multiply(a, b) { return a * b; } const operation = multiply; console.log(operation(4, 5));

7. Functions are Objects

JavaScript mein functions bhi objects hote hain.

Inke paas properties aur methods hote hain jaise .name, .length, .toString().

function demo(x) { return x * x; } console.log(demo.name); // Output: demo console.log(demo.length); // Output: 1

8. Arrow Functions

ES6 mein introduce hui thi.

Short syntax aur lexical this binding hoti hai.

function keyword nahi hota.

Syntax: const functionName = (parameters) => { // Code to be executed }; 

                const add = (a, b) => a + b;
                console.log(add(2, 3));
